# ACGCrawlSpider
爬取 18h.animezilla.com 网站的漫画,

该爬虫是入门级, 适合初学者食用(学习娱乐两不误)



**主要功能:**

1. 指定漫画id爬取
2. 爬取全部高质量漫画
3. 制定目录下载
4. 漫画自动按漫画名分类存储
5. 支持漫画级别去重和图片级别重爬



**环境:** python3.7 

> 已提供 requirments.txt ,最重要的爬虫包和图片包:(Scrapy, Pillow)



**项目运行:**

​	执行 startACG.py 文件即可



**项目盘点:**

1. 此网站需有梯子才能访问
2. 有简单的防爬机制(需要制定referer)
3. 整个网站规整,扩展该脚本也很容易



**适用人群:**

1. 有一定的python基础
2. 有梯子(这个网站是国外网站,除肉身在外者)
